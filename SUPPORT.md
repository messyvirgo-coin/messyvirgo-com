# Support

This repository hosts the public website for Messy Virgo (`messyvirgo.com`).

## Where to ask questions

- **Website**: check the site first for documentation and updates.
- **GitHub issues**: use issues for bugs and concrete, actionable improvements to the website/repo.

## Security issues

Please **do not** open public issues for suspected vulnerabilities. Report them privately to:
